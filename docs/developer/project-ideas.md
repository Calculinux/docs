# Project Ideas

While we have limited RAM and CPU resources, there's a surprising account you can do with 128 MB of RAM. Terminal based productivity apps have been around since the dawn of computing. For anything you decide to work on, you could choose to port something that exists, customizing and creating a package, or you could create an entirely new application to take full advantage of the Picocalc's unique capabilities!

## Basic Productivity Project Ideas

| Name          | Description                                                                           | Examples                           |
|---------------|---------------------------------------------------------------------------------------|------------------------------------|
| Spreadsheets  | An app to organize data into endless rows and columns, with formulas for convenience. | Microsoft Excel, LibreOffice Calc  |
| RSS Reader    | Collect and read RSS feeds from multiple sources in one easy-to-use interface.        | Newsboat, Canto                    |
| Email Grabber | Retrieve emails from Outlook, Gmail, etc., and bring them to the CLI!                 | Snarf, Sky Email Extractor, Alpine |
