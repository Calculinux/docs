# Project Ideas

So, you like to code, right? And you have some free time? Well… why not support the PicoCalc community by contributing to any of the project ideas below to make the PicoCalc more usable as an everyday computer for everyday people?

## Urgent

### Not urgent, but high-priority projects

| Name          | Description                                                                 | Examples                          |
|---------------|-----------------------------------------------------------------------------|-----------------------------------|
| Spreadsheets  | An app to organize data into endless rows and columns, with formulas for convenience. | Microsoft Excel, LibreOffice Calc |
| RSS Reader    | Collect and read RSS feeds from multiple sources in one easy-to-use interface. | Newsboat, Canto                   |
| Email Grabber | Retrieve emails from Outlook, Gmail, etc., and bring them to the CLI!        | Snarf, Sky Email Extractor        |

## Not Urgent

| Name              | Description                                                                 | Examples              |
|-------------------|-----------------------------------------------------------------------------|-----------------------|
| Game Boy Emulator | What did you expect? It’s for playing games, obviously!                     | mGBA, SameBoy         |
| Text Messaging    | Send and receive text messages.                                             | Irssi, Profanity      |
| Docupedia         | A place where you can find documentation for almost anything.               | Wikipedia, Grokipedia |
| Web Browser       | A text-based web browser designed for efficient everyday web browsing.      | w3m, Lynx             |

If you have any other project ideas, feel free to start working on them to make Calculinux better, and post your ideas on the Calculinux megathread!
