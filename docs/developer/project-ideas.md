# Project Ideas
So, you like to code, right? And, you have some free time? Well... why not support the PicoCalc community, by contributing to any of the project ideas below, to make the PicoCalc more usable as an everyday
computer, for everyday people?

## Urgent
##### Not Urgent, but high priority projects
|Name|Description|Examples|
|----|-----------|--------|
|Spreadsheets|App to organise data into endless rows and colums with formulas for convinience.|Microsoft Excel, Libre Office Calc|
|RSS Reader|To collect and read RSS feeds from multiple places, and put them in one easy interface.|Newsboat, Canto|
|Email Grabber|To grab your emails from Outlook, Gmail, etc, and bring them on CLI!|Snarf, Sky Email Extractor|

## Not Urgent
|Name|Description|Examples|
|----|-----------|--------|
|Gameboy Emulator|What did you expect? It's for playing games obviously!|mGBA, SameBoy|
|Text Messaging|To message people in texts?|Irssi, Profanity|
|Docupedia?|Literally a place where you would be able to find documentation for almost anything?|Wikipedia, Grokipedia|
|Web Browser|A text-based web browser, to suit everyday web browsing needs, while being efficient.|w3m, Lynx|

If you have any other project ideas, just start working on them, to make calculinux better, and post your ideas on the Calculinux megathread!
